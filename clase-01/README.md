## 2 maneras de ejecutar nuestros scripts en Visual Studio Code

1. Hacer click en el icono de play arriba a la derecha en los archivos .py

2. Abrir la terminal (Ctrl + ñ) y ejecutrar los siguientes comandos

```sh
py <nombre-script.py>
py main.py
python main.py
```
